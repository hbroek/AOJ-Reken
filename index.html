Implemented with web dev framework Reken

Advent of Javascript challenge

Complete 24 Javascript coding challenges from December 1st thru 24th. Organized by selfteach.me The challenges are focussed at making already provided static web sites/apps interactive.

Why Reken?

Reken is an easy way to add functionality to an existing website. I build Reken with that in mind. Take existing HTML sprinkle some Reken data- attributes and see it come alive like putting lights in a Christmas tree.

Who are you?

Glad you asked I'm Henry van den Broek, developer of Reken. And I love this challenge to put Reken through its paces. And sofar with great success. You'll see that some solutions are literally just a sprinkle of Reken goodness.
